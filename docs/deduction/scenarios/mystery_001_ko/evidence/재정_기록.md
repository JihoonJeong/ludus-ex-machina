# 재정 기록 — 용의자

**출처:** 금융감독원 조회, 서울중앙지방법원 영장 발부

## 용의자 A (한지연)
- 월급: 280만 원 (갤러리 마루)
- 예금 잔고: 1,200만 원
- 미상환 채무 없음
- 최근 6개월간 이상 거래 내역 없음
- **판정: 재정 상태 안정적. 금전적 압박 요인 없음.**

## 용의자 B (신원재)
- 자영업 미술품 딜러. 수입 불규칙.
- **미상환 채무: 3억 2천만 원**, 채권자 3곳
  - 하나은행 사업자 대출: 1억 5천만 원 (3개월 연체)
  - 사채업자 (성명 비공개): 1억 2천만 원 (2월 28일 자 독촉장 발송)
  - 신용카드 부채: 5천만 원 (4개 카드 합산)
- 최근 거래 내역:
  - 2월 15일: 2천만 원 현금 인출, 용도 불명
  - 3월 5일: 3천만 원 수표 부도
  - **3월 8일: 사채업자로부터 문자 수신 — "마지막 경고. 3월 15일까지 갚지 않으면 각오해라."**
- **판정: 심각한 재정 위기. 채권자의 위협을 받는 상태. 변제 기한이 범행일로부터 5일 후.**

## 용의자 C (박민수 박사)
- 고려대학교 미술사학과 교수
- 월급: 650만 원
- 투자 자산: 21억 원 (주식, 채권, 부동산)
- 미술품 컬렉션 보험 평가액: 48억 원
- 채무 없음
- **판정: 매우 부유함. 도난에 대한 금전적 동기 없음.**
