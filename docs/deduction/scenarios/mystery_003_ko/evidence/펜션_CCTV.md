# 펜션 CCTV — 파인밸리 펜션, 가평

**카메라 시스템:** 외부 카메라 2대만 운영. 내부 카메라 없음 (펜션 개인정보 보호 방침).

## 카메라 1: 정문 입구 / 주차장
정문, 진입로, 주차 공간을 촬영한다. 후면 데크 또는 온수 욕조 구역은 **촬영 범위에 포함되지 않음**.

| 시각 | 사건 |
|------|------|
| 15:02 | 일행 도착 — 차량 2대 (김현우의 아우디 Q7, 박태현의 테슬라 모델 Y). 5명 전원 펜션으로 들어가는 모습 확인. |
| 15:10 | 차량 1대 추가 도착 (서유나의 현대 아이오닉 6). 서유나 혼자 펜션으로 들어감. |
| 18:15 | 박태현, 차에서 물건을 가져오기 위해 잠시 외출. 3분 후 귀환. |
| **23:25** | **서유나가 정문으로 나옴.** 본인 차량(아이오닉 6)으로 걸어감. 트렁크를 열고 **작은 천 주머니/파우치**를 꺼냄. 2분 이내에 돌아옴. |
| 00:43 | 박태현이 정문으로 나옴. 주차장으로 걸어감. 차 문을 열었다 닫음 (실내등 확인). 00:52에 건물 쪽으로 돌아옴. **참고: 박태현은 건물 오른편을 따라 걷는데, 이 경로가 데크 근처를 지나치나 이 카메라에는 포착되지 않음.** |
| 00:54 | 박태현이 정문으로 들어옴. |

**00:54 이후 06:48 (최은지 아침 산책을 위해 나옴) 사이에 외부 움직임 포착 없음.**

## 카메라 2: 후면 정원 / 데크 부분 촬영
정원 통로와 후면 데크의 약 30%를 촬영한다. 온수 욕조는 데크 끝에 위치하며 **카메라 촬영 범위에 포함되지 않음**. 단, 데크 난간과 계단은 부분적으로 보인다.

| 시각 | 사건 |
|------|------|
| 00:28 | 피해자(정태양)가 유리잔을 들고 데크로 나옴. 온수 욕조 방향으로 걸어감 (카메라 앵글 밖으로 사라짐). |
| 00:47 | 박태현이 건물 오른쪽 방향으로 걷다가 프레임 끝에 잠깐 보임. |
| 00:52 | 박태현이 같은 경로로 돌아옴. 데크 근처에서 잠시 멈춤 (부분 포착). 건물 정면 방향으로 계속 걸어감. |
| **01:18** | **데크 계단 근처에서 인물이 잠깐 보임.** 화면이 어두워 인물은 약 3초 동안만 프레임을 통과한다. **성별, 의복, 신원 확인 불가.** 이동 방향: 건물 내부에서 데크 방향, 이후 온수 욕조 쪽으로 프레임 밖. |
| **01:22** | **동일하거나 다른 인물이 반대 방향으로 프레임을 통과함** (데크에서 건물 내부 방향). 마찬가지로 신원 확인 불가. 지속 시간: 약 2초. |
| 06:48 | 최은지가 아침 산책을 위해 후문으로 나옴. |
| 07:08 | 최은지가 달려서 들어옴. |
| 07:10 | 최은지와 박태현이 데크의 온수 욕조 근처에 있음. |

**수사관 의견:** 01시 18분/22분 영상은 아직 해결되지 않은 가장 핵심적인 요소다. 펜션의 후면 카메라는 저해상도 기기(720p)로 야간 시야가 불량하다. 디지털포렌식수사대의 화질 향상 시도에서도 추가 식별 특징을 얻지 못하였다. 해당 시각은 추정 사망 시간대(오전 1시~3시) 안에 포함된다. 4분간의 공백은 누군가 온수 욕조 구역에 갔다가 돌아왔음을 시사한다.**
