# 알리바이 — 용의자 A (최은지)

**진술 요약:** 0시 30분경 2층 2호실로 들어갔다. 졸피뎀 10mg(처방받은 수면제)을 복용하였다. 오전 1시 이전에 잠들었다. 오전 6시 50분에 기상. 약 7시 10분에 시신을 발견하였다.

**검증:**
- 객실 배정 확인: 2호실, 2층, 창문은 정면 방향 (데크/온수 욕조 반대편)
- 졸피뎀 처방전: 연세세브란스병원 확인 완료. 신경미 박사가 2026년 2월 20일 졸피뎀 10mg 30정을 처방. 약국 조제 기록 확인 완료.
- 소지품 검사: 최은지의 세면도구 가방에서 졸피뎀 병 발견. **27정 남음** (1개월간 간헐적 사용과 일치, 3정 복용)
- 휴대폰 사용 기록: 마지막 사용은 0시 38분 (오전 6시 45분 알람 설정). 오전 6시 48분까지 활동 없음 (알람 해제).
- 객실 상태: 침대 사용 흔적 뚜렷. 밤중 외출 흔적 없음.

**평가: 알리바이 일치. 졸피뎀은 복용 후 20~30분 이내에 깊은 수면을 유발한다. 휴대폰 비활성 기록이 오전 1시~6시 48분 수면을 뒷받침한다. 최은지가 중요한 시간대에 객실을 떠났다는 증거 없음.**
