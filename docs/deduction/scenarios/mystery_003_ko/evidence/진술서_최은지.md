# 진술서 — 용의자 A (최은지)

**일시:** 2026년 3월 22일 오전 10시
**장소:** 가평경찰서

저희는 21일 오후 3시쯤 펜션에 도착했어요. 분위기가 좋았어요 — 18년 동안 이어온 연례 행사거든요. 저녁 6시쯤부터 고기를 굽고, 8시쯤 거실로 들어와 술을 마셨어요.

태양이는 기분이 아주 좋았어요. 세종시 개발 사업을 방금 따냈다고 좋아하더라고요. 평소보다 많이 마셨어요 — 주로 위스키를.

밤 10시쯤 태양이와 앞마당에서 단둘이 이야기를 했어요. 태양이가 먼저 대학 시절 이야기를 꺼냈고, 감상적이 됐어요. 우리 사이가 그렇게 끝난 게 아쉽다고 했어요. 저는 23년 전 일이고, 우리 둘 다 잘 살고 있잖냐고 했어요. 좀 어색하긴 했지만 감정적이진 않았어요. 현우가 우리가 이야기하는 걸 보고 나중에 무슨 얘기냐고 물었는데, 그냥 옛날 추억 이야기라고 했어요.

다섯 명이 자정쯤까지 마셨어요. 0시 15분경 태현이가 이제 자자고 했어요. 태양이는 온수 욕조에 좀 더 있겠다고 했어요 — "별 보면서 있고 싶다"고 했죠. 위스키 잔을 들고 데크로 나갔어요.

저는 0시 30분경 방(2층 2호실)으로 들어갔어요. **수면제를 먹었어요** (늘 먹는 졸피뎀 10mg, 담당 의사한테 처방받은 거예요) — 낯선 침대에서 잠을 잘 못 자거든요. 1시 전에 잠들었어요.

새벽 6시 50분쯤 일어나서 산책하러 나갔는데, 그때 태양이를 온수 욕조에서 발견했어요. 뭔가 잘못됐다는 걸 바로 알았어요 — 얼굴이 물속에 잠겨 있었으니까요. 비명을 질렀고 태현이가 뛰어나왔어요. 끌어냈지만 이미 몸이 차가웠어요.

**유나 씨에 대해서** — 어젯밤엔 평소보다 더 말이 없었어요. 원래 조용한 편이긴 한데, 뭔가 딴 데 정신이 팔린 것 같았어요. 모두의 잔을 계속 채워주던 건 친절하다고 생각했어요. 일찍, 아마 11시 30분쯤 방에 들어갔어요. 가평까지 운전해 와서 피곤하다고 했어요.

왜 누군가가 태양이를 해치려 했을지 모르겠어요. 거만하다는 말을 들을 수는 있었지만, 어쨌든 우리 친구였는데요.
