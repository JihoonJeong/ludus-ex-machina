# 금전 분쟁 — 정태양 & 김현우

**출처:** 김현우 제공 문서 및 독립 검증 완료

## 배경

2025년 1월, 김현우(미래에셋증권 경유)는 세 명의 개인 투자자로부터 피해자의 회사인 한강개발(주)이 시행하는 위례 신도시 부동산 개발 사업에 30억 원을 투자받도록 주선하였다. 김현우는 피해자와의 친분을 근거로 투자자들에게 해당 투자를 개인 보증하였다.

## 분쟁 경과

| 날짜 | 사건 |
|------|------|
| 2025년 1월 | 30억 원 투자 확정. 준공 목표: 2025년 12월. |
| 2025년 6월 | 사업 지연 — 인허가 문제. 새 목표: 2026년 3월. |
| 2025년 9월 | 2차 지연 — 공사비 초과. 김현우, 투자자들의 불만을 받기 시작. |
| 2025년 12월 | 당초 준공 기한 경과. 피해자로부터 진행 상황 업데이트 없음. |
| 2026년 1월 | 김현우, 공식 사업 현황 보고서 요청. 피해자, 막연한 답변만 제공. |
| 2026년 2월 | 투자자 중 한 명이 김현우를 상대로 법적 조치 위협. |
| 2026년 3월 13일 | 김현우, 법무법인 김앤파트너즈를 통해 정식 내용증명 발송: 30일 이내 30억 반환 또는 중재 진행. |
| 2026년 3월 15일 | 모임 당일 언쟁 발생. |

## 재정적 위험

- 김현우의 개인 보증 책임: 최대 30억 원
- 김현우의 연간 소득: 약 4억 원 (급여 + 성과급)
- 투자자들이 소송을 제기할 경우 개인 파산 가능성
- 법무법인 김앤파트너즈가 3월 13일 내용증명 발송 사실 확인

## 동기 평가

김현우에게는 피해자에 대한 강한 재정적 불만이 있었다. 그러나:
- 피해자의 사망이 김현우의 재정 문제를 **해결하지 않는다** — 30억은 여전히 사업에 묶여 있다.
- 한강개발 이사회가 사업의 향후 방향을 결정해야 한다. 피해자의 사망은 자금 회수를 단순하게 하는 것이 아니라 오히려 **복잡하게 만들 수 있다**.
- 김현우가 이미 진행 중이던 전략(법적 중재)은 충분한 승산이 있었다.
- **피해자를 죽이면 자금 회수가 더 쉬워지는 것이 아니라 더 어려워진다.**

**평가: 감정적 동기는 강하나(분노, 신뢰 배신) 합리적 동기는 약하다. 피해자의 사망은 김현우에게 재정적으로 역효과를 낳는다.**
