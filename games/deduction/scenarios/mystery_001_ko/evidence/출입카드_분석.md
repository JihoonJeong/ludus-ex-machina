# 출입카드 포렌식 분석

## 카드 KC-0447 — 기술 보고서

IT 보안팀이 출입 통제 시스템에 기록된 카드 KC-0447의 암호화 서명을 분석하였다.

### 분석 결과

1. **카드 규격:** NovaTech 출입 시스템과 호환되는 표준 HID iCLASS SE 규격.

2. **암호화 키:** 해당 카드는 NovaTech의 시설 마스터 키를 사용하고 있으며, 이는 NovaTech 자체 카드 프로그래밍 장비(또는 마스터 키 복제본)로 발급되었음을 입증한다.

3. **카드 일련번호 출처:** KC-0447의 내부 일련번호는 KC-0100부터 KC-0150까지의 제조 배치 접두사를 공유한다. 이 배치는 2025년 1월부터 6월 사이에 NovaTech 직원에게 발급된 것이다.

4. **복제 흔적:** 해당 카드의 섹터 배열은 카드 **KC-0112**와 동일하다. KC-0112는 이전에 **권대위드(전 선임 엔지니어)**에게 발급된 카드다. KC-0112는 2026년 1월 15일 권 씨의 해고 시 출입 시스템에서 비활성화되었다. 그러나 KC-0447은 ID 섹터에 다른 카드 번호가 기록된 KC-0112의 **물리적 복제본**으로 확인된다.

5. **프로그래밍 타임스탬프:** 카드 내부 프로그래밍 타임스탬프는 **2026년 2월 28일**을 가리킨다 — 권 씨의 해고로부터 약 6주 후이다.

### 결론

카드 KC-0447은 권대위드의 이전 출입카드 KC-0112의 복제본이다. 그의 해고 이후에 HID iCLASS SE 카드의 읽기/쓰기가 가능한 장비를 사용하여 제작되었다. 재직 기간 동안 권 씨는 시설 관리 업무의 일환으로 IT실의 카드 프로그래밍 장비에 대한 접근 권한을 보유하고 있었다.
